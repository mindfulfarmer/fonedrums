FoneDrums is a lightweight, low-latency web application designed to turn your mobile device or desktop into a playable drum pad. Unlike typical web drum kits that play pre-recorded MP3 samples, FoneDrums uses the Web Audio API to synthesize sounds in real-time, allowing for dynamic sound shaping, zero file-loading lag, and a tiny memory footprint.
